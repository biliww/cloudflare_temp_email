## Naive UI 配置说明

### 字体相关
- **fontFamily**: 设置常规字体族，包括 `v-sans` 和多个系统字体，作为备用字体。
- **fontFamilyMono**: 设置等宽字体族，通常用于代码或固定宽度文本。
- **fontWeight**: 字体的常规权重，400 通常代表普通体重。
- **fontWeightStrong**: 强调字体的权重，500 一般表示加粗。

### 动画相关
- **cubicBezierEaseInOut**: 过渡动画的缓动函数（从开始到结束的平滑变化）。
- **cubicBezierEaseOut**: 动画结束时的缓动函数（逐渐减速）。
- **cubicBezierEaseIn**: 动画开始时的缓动函数（逐渐加速）。

### 边框与圆角
- **borderRadius**: 设置普通元素的圆角大小。
- **borderRadiusSmall**: 设置较小的圆角，通常用于按钮或卡片。

### 字体大小
- **fontSize**: 正文字体大小，14px。
- **fontSizeMini**: 最小字体大小，12px。
- **fontSizeTiny**: 很小的字体大小，12px。
- **fontSizeSmall**: 较小的字体大小，14px。
- **fontSizeMedium**: 中等字体大小，14px。
- **fontSizeLarge**: 较大字体大小，15px。
- **fontSizeHuge**: 很大的字体大小，16px。

### 行高与高度
- **lineHeight**: 设置文本行高，1.6 表示行高是字体大小的 1.6 倍。
- **heightMini**: 最小高度，16px。
- **heightTiny**: 很小的高度，22px。
- **heightSmall**: 较小的高度，28px。
- **heightMedium**: 中等高度，34px。
- **heightLarge**: 较大高度，40px。
- **heightHuge**: 最大高度，46px。

### 颜色
- **baseColor**: 基本背景颜色，白色 (`#FFF`)。
- **primaryColor**: 主要颜色，绿色 (`#18a058`)。
- **primaryColorHover**: 主要颜色悬停状态的颜色，稍微明亮的绿色。
- **primaryColorPressed**: 按下时的主要颜色，深绿色。
- **primaryColorSuppl**: 补充主要颜色，明亮绿色。
- **infoColor**: 信息颜色，蓝色 (`#2080f0`)。
- **successColor**: 成功状态的颜色，绿色 (`#18a058`)。
- **warningColor**: 警告状态的颜色，橙色 (`#f0a020`)。
- **errorColor**: 错误状态的颜色，红色 (`#d03050`)。

### 文本颜色
- **textColorBase**: 默认文本颜色，黑色 (`#000`)。
- **textColor1**: 次级文本颜色，深灰色 (`rgb(31, 34, 37)`)。
- **textColor2**: 较浅的文本颜色，灰色 (`rgb(51, 54, 57)`)。
- **textColor3**: 说明文本的颜色，更浅的灰色 (`rgb(118, 124, 130)`)。
- **textColorDisabled**: 禁用文本的颜色，灰白色 (`rgba(194, 194, 194, 1)`)。

### 图标颜色
- **iconColor**: 默认图标颜色，浅灰色 (`rgba(194, 194, 194, 1)`)。
- **iconColorHover**: 图标悬停时的颜色，稍深的灰色 (`rgba(146, 146, 146, 1)`)。
- **iconColorPressed**: 图标按下时的颜色，暗灰色 (`rgba(175, 175, 175, 1)`)。

### 透明度
- **opacity1**: 透明度 0.82。
- **opacity2**: 透明度 0.72。
- **opacity3**: 透明度 0.38。
- **opacity4**: 透明度 0.24。
- **opacity5**: 透明度 0.18。

### 分隔线与边框
- **dividerColor**: 分隔线颜色，浅灰色 (`rgb(239, 239, 245)`)。
- **borderColor**: 边框颜色，浅灰色 (`rgb(224, 224, 230)`)。

### 按钮与图标
- **closeIconColor**: 关闭图标的颜色，灰色 (`rgba(102, 102, 102, 1)`)。
- **closeIconColorHover**: 关闭图标悬停时的颜色，灰色 (`rgba(102, 102, 102, 1)`)。
- **closeIconColorPressed**: 关闭图标按下时的颜色，灰色 (`rgba(102, 102, 102, 1)`)。

### 滚动条相关
- **scrollbarColor**: 滚动条的颜色，透明黑色 (`rgba(0, 0, 0, 0.25)`)。
- **scrollbarColorHover**: 滚动条悬停时的颜色，透明黑色 (`rgba(0, 0, 0, 0.4)`)。
- **scrollbarWidth**: 滚动条宽度，5px。
- **scrollbarHeight**: 滚动条高度，5px。
- **scrollbarBorderRadius**: 滚动条边框圆角，5px。

### 其他样式
- **boxShadow1**: 较轻的阴影效果，用于按钮等元素。
- **boxShadow2**: 中等强度的阴影效果，通常用于卡片或对话框。
- **boxShadow3**: 强烈的阴影效果，适用于深层次的浮动元素。

### 色调
- **hoverColor**: 悬停时的颜色，浅灰色 (`rgb(243, 243, 245)`)。
- **pressedColor**: 按下时的颜色，灰白色 (`rgb(237, 237, 239)`).
- **inputColorDisabled**: 禁用输入框颜色，灰白色 (`rgb(250, 250, 252)`).
- **buttonColor2**: 第二种按钮颜色，带透明背景。
